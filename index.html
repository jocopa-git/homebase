Setup Guide

1
Go to console.cloud.google.com → enable Google Calendar API.
2
Create OAuth client ID (Web application). Add your HomeBase URL as both origin and redirect URI.
3
Copy the Client Secret from the OAuth credentials page. In Netlify: Site settings → Environment variables → add GCAL_CLIENT_SECRET with the secret as the value.
4
Copy the Client ID above and click Sign in with Google. You only need to do this once per device.

How it works: After signing in, HomeBase receives a refresh token that it stores locally. This token is used to silently get new access tokens every hour — no page redirects, no user interaction, indefinitely. The client secret is stored securely in Netlify and never exposed to the browser.
